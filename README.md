# Laravel Admin CMS

Custom-built Content Management System with authentication from scratch.

## Features
- **Custom Authentication** – Built without scaffolding; session management, middleware protection, "remember me" functionality
- **Dynamic Content Management** – Update hero sections, headings, images on the fly
- **Security-First** – Server-side validation, secure file uploads, protected routes
- **UX Polish** – SweetAlert popups for feedback, graceful error handling
- **Edge Case Handling** – Auto-logout and redirect when admin account is deleted mid-session

## Tech Stack
- Laravel 10.x
- PHP 8.x
- MySQL
- Blade Templates
- JavaScript

## What I Learned
- How Laravel's authentication system works under the hood
- Middleware implementation for role-based access
- Database migrations and seeders for version control
- Handling file uploads securely with validation
